Robin's Default Page on dev00102 WSL

Willkommen auf meiner Apache-Webseite!

Diese Webseite wird unter Windows Subsystem for Linux (WSL) mit einer Debian-Distribution und dem Apache-Webserver betrieben. WSL ermöglicht es, Linux-Tools und -Dienste nahtlos auf einem Windows-System auszuführen, wodurch ich die Flexibilität von Linux und die Benutzerfreundlichkeit von Windows optimal nutzen kann.

Technische Details:

  • Betriebssystem: Debian unter WSL
  • Webserver: Apache HTTP Server
  • Entwicklung: Kombination aus leistungsstarken Linux-Werkzeugen und flexibler Webentwicklung

Warum diese Umgebung?

Die Kombination aus WSL und Debian bietet eine ideale Plattform für die Entwicklung und das Hosting von Webanwendungen in einer Linux-nahen Umgebung, ohne auf die Vorteile eines Windows-Systems verzichten zu müssen.

Linux Testkonfiguration

Apache Webserver

Der Apache-Webserver ist eine leistungsstarke und flexible Plattform für das Hosting von Webseiten. In einer Linux-Testumgebung kann er verwendet werden, um sichere Verbindungen über HTTPS einzurichten und Module wie mod_rewrite oder mod_security zu testen.

Tomcat Server

Apache Tomcat dient als Servlet-Container für Java-Anwendungen. Er eignet sich ideal, um REST-APIs und Java-basierte Webanwendungen zu hosten. Die Konfiguration von SSL und die Integration mit Apache erleichtern den sicheren Betrieb.

Datenbankserver

MySQL: MySQL ist eine weit verbreitete relationale Datenbank, ideal für transaktionsbasierte Anwendungen. In einer Testumgebung kann es zur Optimierung von Abfragen und Backup-Strategien verwendet werden.
PostgreSQL: PostgreSQL bietet erweiterte Funktionen wie JSON-Unterstützung und eignet sich für komplexe Datenmodelle. Es wird häufig zur Optimierung der Leistungsfähigkeit und Skalierbarkeit getestet.

Sicherheitskonfiguration

Wazuh: Wazuh ist eine Open-Source-Lösung zur Sicherheitsüberwachung, die Angriffe erkennt und Abweichungen in der Konfiguration meldet.
Lynis: Lynis ist ein Audit-Tool für Linux-Systeme, das Schwachstellen identifiziert und Best-Practice-Empfehlungen gibt.
auditd: auditd ermöglicht die Überwachung sicherheitskritischer Ereignisse und bietet detaillierte Berichte über Systemaktivitäten.

Eine Kombination dieser Tools und Plattformen sorgt für eine stabile und sichere Linux-Testumgebung, die sowohl Webentwicklern als auch Administratoren optimale Bedingungen bietet.

Softwareentwicklung

Java und Java EE bzw. Spring Boot

Java ist eine robuste Programmiersprache, die sich hervorragend für serverseitige Anwendungen eignet. Java EE erweitert Java um Werkzeuge für Unternehmensanwendungen, wie Servlets, JSP und EJB, und ermöglicht die Entwicklung skalierbarer Systeme.

Spring Boot vereinfacht die Entwicklung von Java-basierten Microservices. Durch automatische Konfiguration und ein breites Ökosystem ist es eine ausgezeichnete Wahl für moderne Webanwendungen.

React & Node.js

React ist eine JavaScript-Bibliothek, die für die Entwicklung von Benutzeroberflächen verwendet wird. Mit ihrer komponentenbasierten Architektur erleichtert sie die Erstellung dynamischer und interaktiver Webanwendungen.

Node.js ist eine serverseitige JavaScript-Laufzeitumgebung, die für ihre hohe Leistung und Skalierbarkeit bekannt ist. Sie eignet sich ideal für die Entwicklung von Echtzeitanwendungen wie Chats oder APIs.
